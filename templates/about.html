Lyrics Decomposed

About

Lyrics Decomposed is a web application that allows users to analyze the topics, polarity and sentiments of song lyrics. This was my capstone project for the Data Science Immersive program at Bitmaker General Assembly. Source code for this project is publicly available on my github.

All the data used to build this app was scraped from Genius.com.

Detailed information on the algorithms and technologies employed can be found in this post I published on my blog.
